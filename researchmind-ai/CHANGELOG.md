# Changelog

## 1.0.0 - 2026-06-27

- Added autonomous LangGraph research workflow.
- Added FastAPI backend with JWT authentication and protected routes.
- Added SQLAlchemy persistence for users, projects, sessions, reports, citations, logs, agent executions, and memory.
- Added ChromaDB memory storage.
- Added PDF export with ReportLab.
- Added request logging, rate limiting, security headers, CSRF checks, metrics, and health endpoints.
- Added Docker, Docker Compose, Nginx, Render, Railway, Alembic, GitHub Actions, and deployment documentation.
